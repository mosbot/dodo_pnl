# Бизнес-контекст компании

> Необязательный файл. Заполняется вручную пользователем.
> Нужен, чтобы агент точнее понимал компанию и лучше интерпретировал цифры из API.

## Зачем этот файл

Данные агент всегда берёт из API ПланФакт. Но одни и те же цифры можно трактовать по-разному в зависимости от бизнеса: для кофейни 2 млн в месяц — это одно, для строительной компании — другое.

Этот файл помогает агенту:

- понимать, что для компании «норма», а что «отклонение»;
- корректнее комментировать сезонность, выручку, расходы, долги и остатки;
- связывать финансовые выводы с масштабом, отраслью и важными KPI;
- не давать общие советы, а говорить в контексте конкретной компании.

## Правила использования

- Файл **необязательный**. Если его нет или он пустой — агент работает только по API и основной документации.
- Если файл заполнен **частично** — агент использует только те факты, которые явно указаны.
- Агент **не додумывает** бизнес-контекст.
- Агент не использует этот файл для выбора endpoint'а, расчёта цифр или заполнения отсутствующих данных.
- Если текст в этом файле расходится с данными API — приоритет у API. API — основной источник фактов, этот файл — только дополнительный контекст для интерпретации.
- Не храните здесь API-ключи, пароли, банковские реквизиты, персональные данные сотрудников или клиентов.

## Минимально достаточно

Чтобы файл уже приносил пользу, хватит заполнить:

- название компании;
- чем занимается;
- отрасль;
- масштаб: сотрудники, филиалы, обороты;
- кто клиенты;
- как компания зарабатывает;
- какие показатели считаются нормой или риском.

Остальное — по желанию.
